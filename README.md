# 🚗 Sistema de Estacionamento

Projeto desenvolvido em C# durante o bootcamp **.NET Developer** da **AVANADE** na plataforma DIO.

---

## O que faz

Gerencia um estacionamento pelo terminal. Você define o preço inicial e o valor por hora, e o sistema cuida do resto — cadastrar veículos, calcular o valor na saída e listar quem está estacionado.

---

## Funcionalidades

- Cadastrar veículo pela placa
- Remover veículo com cálculo automático do valor cobrado
- Listar todos os veículos no estacionamento
- Validação de placa — avisa se o veículo não está cadastrado

---

## Como rodar

Precisa ter o [.NET 6+](https://dotnet.microsoft.com/download) instalado.

```bash
git clone https://github.com/anderson213/SISTEMA-DE-ESTACIONAMENTO-.git
cd SISTEMA-DE-ESTACIONAMENTO-/DesafioFundamentos
dotnet run
```

---

## Exemplo de uso

```
Seja bem vindo ao sistema de estacionamento!
Digite o preço inicial: 5
Agora digite o preço por hora: 2

1 - Cadastrar veículo
2 - Remover veículo
3 - Listar veículos
4 - Encerrar
```
